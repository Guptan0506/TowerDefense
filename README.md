TOWER DEFENSE GAME

A simple tower defense game built using Jawa swing.

Features:
- Main Menu: Start and Quit options
- Wave System: 5 Waves
- Towers:
  - Placeable only on grass
  - Range detection with a hover circle
- Enemies:
  - Move along a winding path
  - Health bars above each one
- Bullets:
  - Glowing Projectile
- Explosions:
  - Animated blast effect when enemies die
- Economy & Lives
  - Earn money for kills
  - Lose lives when enemies escape
- Game Over / You Win screens
  - Restart button to replay after finishing

Technologies Used:
- Java
- Swing for GUI and rendering
- Java2D Graphics for custom drawing and animations

How to run:
- Clone or download this repository

Possible future improvements:
- Tower upgrades
- Sound effects and music
- Pause menu
